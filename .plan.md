# Plan: Luxury Futuristic Redesign

## Goal
Elevate the site's visual identity toward **dark luxury tech** — think Linear, Vercel, Apple Pro — while staying minimalist, elegant, and fully mobile-friendly.

## Design Language
- **Base**: Deep void black (`#050507`) with warm zinc text
- **Accents**: Very desaturated aurora — subtle indigo/violet/cyan glows used sparingly on hover states, hero ambient light, and active indicators
- **Texture**: Subtle SVG noise grain overlay (opacity 0.03) for tactile luxury
- **Glass**: Backdrop blur on nav and cards with 1px `zinc-800/30` borders
- **Motion**: Smooth fade-up entrances, hover glows, ambient breathing gradients
- **Typography**: Keep Geist. Hero at `text-5xl`→`text-7xl`, tighter tracking (`tracking-tighter`), generous line-height

## Implementation Details

### 1. `globals.css` — Theme Engine
- Define CSS custom properties for accent gradient colors (indigo/violet/cyan at very low saturation)
- Add `@keyframes` for:
  - `fade-up`: translateY(20px)→0 + opacity
  - `aurora-shift`: slow gradient position drift
  - `glow-pulse`: subtle border glow breathing
- Add `.animate-fade-up` utility with stagger support via inline `--delay`
- Add global `::selection` (already good)
- Keep Tailwind v4 `@theme inline` structure intact

### 2. `layout.tsx` — Global Atmosphere
- Change body bg to `#050507` (deeper than current `#09090b`)
- Add a fixed `<div>` with SVG noise texture (`data:image/svg+xml;base64,...`) at pointer-events-none for grain
- Ensure `scroll-smooth` and `antialiased` remain

### 3. `Navigation.tsx` — Glass Command Bar
- Refine scrolled state: `bg-zinc-950/70 backdrop-blur-xl border-zinc-800/40` (stronger blur, more refined border)
- Add active link indicator: a tiny 4px dot or bottom border in accent color
- Mobile menu: animate height/opacity instead of instant show/hide; maybe use max-height transition
- Logo: add a subtle hover glow
- Ensure `z-50` and fixed positioning remain

### 4. `HeroSection.tsx` — The Statement
- Replace single radial glow with a multi-layer aurora:
  - Large soft indigo/violet blob top-left
  - Small cyan blob bottom-right
  - Both with heavy blur (120px+) and very low opacity (0.15)
- Add an animated aurora shift via CSS
- Typography:
  - Name: `text-5xl sm:text-6xl md:text-7xl font-semibold tracking-tighter`
  - Subtitle: `text-lg sm:text-xl text-zinc-400 max-w-2xl`
- Social links: replace text with small icon-like circles or keep text but add underline animation on hover
- Buttons:
  - Primary: `bg-zinc-100 text-zinc-950 hover:bg-white hover:shadow-[0_0_20px_rgba(255,255,255,0.15)]`
  - Secondary: `border-zinc-700 hover:border-zinc-500 hover:bg-zinc-800/30`
- Entrance: wrap elements in `.animate-fade-up` with staggered delays

### 5. `AboutSection.tsx` — Refined Grid

### 8. `ContactSection.tsx` — Premium CTA

### 9. `Footer.tsx` — Minimal Finish
- Keep border-top `border-zinc-800/40`
- Reduce vertical padding slightly: `py-8`
- Links: add hover underline effect

## Mobile Considerations
- All aurora blobs use `max-w`/`max-h` to avoid overflow issues on small screens
- Hero text scales: `text-5xl` on mobile, `md:text-7xl`
- Navigation hamburger menu gets smooth `max-height` transition instead of instant toggle
- Touch targets remain ≥44px
- Grid collapses to 1-col on mobile naturally via existing `grid-cols-1`
- No hover-dependent info (everything reveals on touch via active states or is always visible)

## Dependencies
No new packages required. Everything achievable with:
- Tailwind CSS v4 utilities
- CSS custom properties and keyframes
- React state for mobile menu
